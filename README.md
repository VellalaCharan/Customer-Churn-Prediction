# Customer Churn Prediction

## Project Overview
This project predicts whether a customer is likely to leave a telecom company using machine learning.

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Telco Customer Churn Dataset

## Project Tasks
- Data Cleaning
- Missing Value Analysis
- Data Visualization
- Customer Churn Analysis
- Machine Learning Prediction using Random Forest

## Conclusion
This project helps identify customers who are likely to churn, enabling businesses to take preventive actions.
